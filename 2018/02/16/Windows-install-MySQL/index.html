在 Windows 环境下安装 MySQL

安装步骤

  1. 解压 mysql-5.7.20-winx64.zip 至 e:\software\mysql

  2. 将e:\software\mysql\bin添加至系统变量path中，使得 cmd 中可以直接运行 bin 文件夹下的程序

  3. 在 e:\software\mysql 目录下新建my.ini，按情况修改配置

  4. 初始化 data 目录

  5. 将mysql安装为windows服务，然后启动mysql
